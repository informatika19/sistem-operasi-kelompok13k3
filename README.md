# sistem-operasi-kelompok13k3

## Aplikasi utilitas
- mkdir (Utilitas untuk membuat direktori baru) 
Command mkdir
```
mkdir new_directory
```
- rm (utilitas untuk menghapus file atau folder)
Command rm file
```
rm file_name
```
Command rm folder
```
rm folder_name
```
- cat (utilitas untuk mencetak isi file)
Command cat
```
cat file_name
```
